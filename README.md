# network-intrusion-detection2


A machine learning project that detects malicious network traffic using the UNSW-NB15 dataset. Trains and compares Logistic Regression, Random Forest, and XGBoost classifiers across 9 attack categories.

**Status:** In progress

---

## Dataset

This project uses the [UNSW-NB15 dataset](https://research.unsw.edu.au/projects/unsw-nb15-dataset) created by the Cyber Security team at the University of New South Wales.

Download `UNSW-NB15_1.csv` through `UNSW-NB15_4.csv` and `UNSW-NB15_features.csv`, then place them in `data/raw/`.
