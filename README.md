# Network Intrusion Detection System

A machine learning project that detects malicious network traffic using the
UNSW-NB15 dataset. Trains and compares Logistic Regression, Random Forest,
and XGBoost classifiers on 2.5 million real network connections.

Built as a portfolio project — Indiana University, Data Science '28

---

## Results

All three models were evaluated on a held-out test set of 508,010 connections.

| Model | Accuracy | F1 Score | Precision | Recall |
|---|---|---|---|---|
| Random Forest | 99.59% | 98.38% | 98.49% | 98.27% |
| XGBoost | 99.49% | 97.96% | 98.23% | 97.70% |
| Logistic Regression | 98.81% | 95.39% | 93.91% | 96.92% |

Random Forest achieved the highest F1 score, correctly identifying 98.27% of
attacks with a false positive rate under 2%. XGBoost trained in under 9 seconds
with nearly identical accuracy — the best speed/performance tradeoff for
real-world deployment.

---

## Dataset

Uses the [UNSW-NB15 dataset](https://research.unsw.edu.au/projects/unsw-nb15-dataset)
created by the Cyber Security Research Group at the University of New South Wales.
2,540,047 network connections labeled across 9 attack categories.

Download `UNSW-NB15_1.csv` through `UNSW-NB15_4.csv` and
`UNSW-NB15_features.csv` and place them in `data/raw/`.

## Usage

Run the pipeline in order:
```bash
# 1. Preprocess the raw data
python src/preprocess.py

# 2. Train all three models
python src/train.py

# 3. Evaluate and generate charts
python src/evaluate.py
```

Or explore interactively in the notebooks:
- `notebooks/01_eda.ipynb` — data exploration and visualizations
- `notebooks/02_modeling.ipynb` — model training, evaluation, and feature importance
